# Full Name Calculator Application

A full-stack web application that functions as a calculator where users can input their full names. The application displays the calculator's result along with the user's name and provides a table to manage all stored names.

## Project Structure

```
sample/
├── frontend/          # ReactJS powered by Vite
├── backend/           # Laravel API
└── README.md          # Project documentation
```

## Features

- **Name Calculator**: Input full name and display calculator result in the same row
- **User Management**: View all users in a table with options to update or delete
- **Input Validation**: Ensures full names contain only alphabetic characters (no numbers)
- **RESTful API**: Full CRUD operations (GET, POST, UPDATE, DELETE)
- **Responsive Design**: Mobile-friendly user interface
- **Error Handling**: Handles invalid inputs and failed API requests
